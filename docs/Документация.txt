Grabber Class:
    Создан класс Grabber, который инициализирует захват видео с помощью OpenCV (cv2.VideoCapture) и предоставляет метод get_image() для получения текущего кадра.
    Инициализация виртуальной камеры OBS:
    Метод obs_vc_init() используется для установки capture_device, который соответствует виртуальной камере OBS (обычно это 0, если это первый подключенный видео источник).
    Захват и отображение видео:
    Внутри основного цикла видео кадры захватываются с помощью метода get_image() и отображаются с помощью cv2.imshow.
